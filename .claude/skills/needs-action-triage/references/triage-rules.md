# Triage rules (Bronze)

## Priority rules

Apply `Company_Handbook.md` if present.
Fallback rules if handbook is missing:
- HIGH: contains (urgent|asap|deadline|payment|invoice|security)
- MEDIUM: normal requests that imply action
- LOW: FYI/newsletters/non-actionable

## Plan quality bar

- Include 3–7 checkboxes.
- Include at least one explicit “Next Steps” line.
- Include a clear “Done condition” statement.

## Archive rules

- Only move to `Done/` after a plan has been created.
- Add `processed:` and `result:` metadata when archiving.
